# Server
## Prerequisite
- Node.js v16.18.0
- A browser (Not IE please)
## Setup
1. Install packages
    ```
    npm install
    ```
## How to run
1. Make sure setup is completed
2. Start the server 
    ```
    npm start
    ```
    Please make sure port 3000 is free.  
    Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
